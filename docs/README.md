## Для кого это руководство

Это руководство рассчитано на начинающих разработчиков. Мы будем объяснять каждую конструкцию языка Go, каждый оператор и ключевое слово. Если вы никогда не писали на Go — не волнуйтесь, мы разберём всё с нуля.

Знание Telegram Bot API не требуется — мы объясним все необходимые концепции по мере изучения материала.

---

## Что вы создадите

По итогам руководства у вас будет:
- Работающий Telegram-бот на Go
- Система обработки команд и сообщений
- Интерактивные клавиатуры и кнопки
- База данных для хранения данных пользователей
- Админ-панель с правами доступа
- Функционал работы с файлами и медиа
- Готовый к развёртыванию проект

---

## Требования

- Компьютер с Windows, macOS или Linux
- Установленный Go (версия 1.21 или выше)
- Учётная запись в Telegram
- Текстовый редактор (рекомендуем VS Code с расширением Go)
- Желание учиться
